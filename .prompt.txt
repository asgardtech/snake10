```typescript
private isBoundaryCollision(pos: Position): boolean {
```

```typescript
if (this.isCollision(newHead) || this.isBoundaryCollision(newHead)) {
```


**Impact**: This is a significant behavioral difference. Wraparound movement (snake reappears on opposite side) is fundamentally different from boundary collision (game ends). The implementation doesn't match the stated feature set.
